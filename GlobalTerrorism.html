Data source¶

The Global Terrorism Database (GTD) is an open-source database that includes information about terrorist attacks around the world from 1970 to 2106. It is maintained by researchers at the National Consortium for the Study of Terrorim and Responses to Terrorim (START), headquartered at the University of Maryland.

Data structure¶

The data is divided into colums that contains informations such as: year, month, day, country, region, city, type, target, killed and wounded due to terrorist attack.

Importing process¶

The data is saved to the csv file and is imported using read_csv method from pandas library. Only columns that contains information about id, year, country, attack type, target, number of fatalities and injuries are used. Characters are encoded using ISO-8859-1 standard. Some rows in cloumns 'fatalities' and 'injuries' are empty and they are filled with 0 value.

Counting a number of terrorist attacks in a different countires¶

Data is grouped by countries and the number of attacks in different countries is count. Then the values are sort from the highest to the smallest and stored into a dictionary.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

terror_data = pd.read_csv('globalterrorismdb_0617dist.csv', encoding='ISO-8859-1', usecols=[0, 1, 8, 29, 35, 98, 101])
    
terror_data = terror_data.rename(columns={'eventid':'id', 'iyear':'year', 'country_txt':'country', 'attacktype1_txt':'attack', 'targtype1_txt':'target', 'nkill':'fatalities', 'nwound':'injuries'})
terror_data['fatalities'] = terror_data['fatalities'].fillna(0).astype(int)
terror_data['injuries'] = terror_data['injuries'].fillna(0).astype(int)

count_country = terror_data.groupby(['country']).count()
count_country = count_country.sort_values('id', 0, False)[:10]
attacks = {count_country.index[i] : count_country.id[i] for i in range(10)}
print(attacks)

plt.figure(num=None, figsize=(13, 6), dpi=80, facecolor='w', edgecolor='k')
plt.bar(range(len(attacks)), attacks.values(), 0.5, color="blue")
plt.xticks(range(len(attacks)), attacks.keys())
plt.xlabel('Country')
plt.ylabel('Terrorist Attacks')
plt.title('Countries with the biggest number of terrorist attacks (1970-2016)')
plt.show()

Counting a number of victims per attack in Iraq depending on an attack type¶

Only the data for Iraq is being analysed. The data is gruped by attack type, count and sotred in dictionary so we have number of attack depending on attack type. Then the data for Iraq is again grouped by attack type, summed and a sum of fatalities and injuries is stored into another dictionary so we have number of victims depending on attack. At the end data for number of victims is divided by data for number of attacks so we have number of victims per attack depending on attack type.

sum_attack_type = terror_iraq.groupby(['attack']).sum()
attack_victims = {sum_attack_type.index[i] : sum_attack_type.fatalities[i] + sum_attack_type.injuries[i] for i in range(len(sum_attack_type)-1)}

victims_per_attack = {count_attack_type.index[i] : round(attack_victims.get(count_attack_type.index[i]) / attack_num.get(count_attack_type.index[i]), 2) for i in range(len(sum_attack_type)-1)}
